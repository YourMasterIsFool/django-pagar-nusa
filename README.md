# Pagar Nusa

## Setup




```bash

# install dependencies
$ pip install -r requirements.txt

# sesuaikan database di app/config/settings.py

# migrate database
$ python manage.py migrate

# serve
$ python manage.py runserver 

# add superadmin
$ python manage.py createsuperuser

# create dummy tier
$ python manage.py seed

# run server

$ python manage.py runserver

```
